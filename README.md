# Challenges-in-addressing-causal-questions-using-administrative-data
Exemplar analyses of simulated data inspired by ECHILD

This depository includes the simulated data and the Stata and R codes used in the paper entitled "Data Note:
Challenges in addressing causal questions using administrative data".

The data are in the Stata file called: "Simulation_timevar_SEND_small.dta" help in the folder called "Data".

the Stata and R codes used to address questions 1-3 (with results reported in Tables 2-4) in the paper are in their respective sub-folders.
Additional "REadMe" files can be found in there with explanations.
